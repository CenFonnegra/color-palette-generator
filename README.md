# Interactive Palette Generator

A simple and interactive web app that generates random color palettes in HEX and HSL format.

## Features
- Generate random color palettes between (6, 8, 9) number of colors.
- Supports HEX and HSL formats.
- Click on a color to copy its HEX code.
- Toast notification for feedback.

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Live Demo
https://cenfonnegra.github.io/color-palette-generator/

## Installation
1. Clone the repo
2. Open `index.html` in your browser
